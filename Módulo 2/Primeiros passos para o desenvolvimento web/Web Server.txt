Web Server

Existem 2 tipos: estático e dinâmico; o estático é um servidor físico onde são armazenados arquivos, softwares e/ou bancos de dados. O dinâmico se refere aos softwares que estão presentes no servidor físico.

Web server dinâmico

Arquivos(file server)
Aplicações(application server)
Banco de dados(database)

Web-service dinâmico:
Um site, ou aplicativo, precisa estar hospedado em um servidor para poder ser acessado. Os dados de um site precisam estar em um servidor. O banco de dados de um site ou aplicativo precisa estar em um servidor.

Exercício final
Tente encontrar, na internet, como se dá o acesso aos web-services dos correios e do governo para, por exemplo, buscar os dados de um CEP e os dados sobre os estudantes do Brasil, respectivamente.
