Roteadores, switches e modems

- Modem (Modulator-demodulator): É o hardware que converte dados em um formato que possa ser transmitido de um computador para outro e lido por outro.

- Roteador: Distribui internet para um ou mais dispositivos de uma rede, pode fazer a comunicação entre redes.

- Switch: Distribui para um ou mais dispositivos de uma rede os dados que chegam de forma correta. Criado para ser "inteligente"
