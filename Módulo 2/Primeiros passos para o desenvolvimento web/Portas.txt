Portas:

As portas são os lugares por onde sairão e chegarão os dados. Em redes de computadores, uma porta é um software de aplicação específica ou processo específico servindo de ponto final de comunicações em um sistema operacional hospedeiro de um computador. Uma porta tem associação com o endereço de IP do hospedeiro, assim como o tipo de protocolo usado para comunicação.
Algumas portas bem conhecidas:

20: FTP
22: SSH
25: SMTP
53: DNS
80: HTTP
443: HTTPS
